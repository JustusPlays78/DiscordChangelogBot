# DiscordBot
This Discord bot is being done for some private things but made it available for everyone.

## Features
- API Endpoints:
  - /add-role: 
    - Method: POST
    - Headers: Content-Type: application/json 
    - Body {
      "guildId": "yourGuildID",
      "userId": "UserID",
      "roleId": "RoleID"
      }
  - /add-role:
    - Method: POST
    - Headers: Content-Type: application/json
    - Body {
      "guildId": "yourGuildID",
      "userId": "UserID",
      "roleId": "RoleID"
      }
